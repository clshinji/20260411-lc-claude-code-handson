# SNS 投稿コンテンツの自動生成

## 概要

「投稿ネタが思いつかない…」「毎日の投稿文を書くのが大変…」そんな悩みを Claude Code が解決します。
楽天ROOM、Instagram、X（旧Twitter）など、どんな SNS でも使える投稿ネタ・文章を一気に生成できます。

---

## 必要な準備

- **Claude Code** がインストール済みであること
- 投稿したい **SNS のジャンルやテーマ** を決めておく（例：美容グッズ、ハンドメイド、カフェ巡り など）

---

## 手順

### Step 1: 作業フォルダを作る

```bash
mkdir ~/Desktop/sns-content && cd ~/Desktop/sns-content
```

### Step 2: Claude Code を起動する

```bash
claude
```

### Step 3: 1週間分の投稿ネタを一気に作る

以下のプロンプトをコピペして、自分の情報に書き換えてから入力してください。

```
私は楽天ROOMで美容グッズを紹介しています。

1週間分（月〜日）の投稿ネタを考えてください。
それぞれの投稿について、以下を作成してください:

- 投稿のテーマ（何について書くか）
- キャッチコピー（思わず読みたくなる一言）
- 投稿文（200文字程度、親しみやすい口調で）
- おすすめハッシュタグ（5個）

結果は見やすい表形式で、sns-plan.md というファイルに保存してください。
```

> Claude Code が 1週間分の投稿プランを自動で作ってくれます！

### Step 4: 朝コレ・0コレ用のテンプレートを作る

楽天ROOM ユーザー向けに、毎日の定型投稿を効率化するテンプレートも作れます。

```
楽天ROOMの「朝コレ」「0コレ」用の投稿テンプレートを作ってください。

- 朝コレ: 毎朝の挨拶 + 今日のおすすめ商品を紹介する投稿
- 0コレ: 深夜0時に投稿する、お得情報やタイムセール商品の紹介

それぞれ5パターンずつ、コピペですぐ使えるようにしてください。
口調は「〜だよ！」「〜してみてね♪」のような親しみやすい感じで。

templates.md に保存してください。
```

### Step 5: Instagram 用のキャプションを作る

Instagram 向けの投稿文も同じように作れます。

```
Instagramでハンドメイドアクセサリーを紹介するアカウントを運営しています。

以下の3つの商品について、それぞれ Instagram 用のキャプションを書いてください:

1. 天然石のピアス（ラピスラズリ）
2. ビーズの編み込みブレスレット
3. レジンの花モチーフネックレス

各キャプションに含めるもの:
- 目を引く1行目（改行なしで表示される部分）
- 商品の魅力を伝える本文（3〜5行）
- おすすめハッシュタグ（10個以上、日本語・英語混合）
- 投稿に最適な時間帯のアドバイス

instagram-captions.md に保存してください。
```

### Step 6: 投稿カレンダーを HTML で作る

作ったネタを見やすいカレンダー形式にまとめることもできます。

```
sns-plan.md の内容をもとに、1週間分の投稿カレンダーをHTMLで作ってください。

- 曜日ごとに投稿内容がカードで表示される
- 各カードに「テーマ」「投稿文」「ハッシュタグ」を表示
- 投稿文はクリックでコピーできるボタン付き
- スマホでも見やすいデザイン
```

```
open calendar.html
```

> ブラウザで投稿カレンダーが表示されます。投稿文をコピーして、そのまま SNS に貼り付けられます！

---

## 応用例

- **月間コンテンツカレンダー** を作って、投稿計画を立てる
- **商品レビューの下書き** を一括生成して、あとから微調整する
- **季節イベント（セール、母の日、クリスマスなど）** に合わせた特別投稿を準備する
- **フォロワーに聞くアンケート投稿** のネタを考えてもらう
- **プロフィール文** を魅力的に書き直してもらう

---

## Tips & コツ

- **自分の口調を教えてあげる** と、より自分らしい文章になります
  ```
  私の口調の特徴: 「〜だよ！」「〜なの♡」をよく使います。絵文字は🌸💕✨が好きです。
  この口調で投稿文を書いてください。
  ```
- **過去のバズった投稿** を参考に見せると、同じテイストで新しいネタを提案してくれます
  ```
  過去にバズった投稿はこれです:「（投稿文をコピペ）」
  同じようなテイストで、新しい投稿を5つ考えてください。
  ```
- **NG ワードを指定** しておくと安心です
  ```
  以下のワードは使わないでください: （薬機法に抵触しそうな表現など）
  ```
- 生成された文章は **必ず自分で確認してから投稿** しましょう。AIが作った文章をそのまま使うと、事実と異なる内容が含まれている可能性があります
