# ホームページ作成

## 概要

Claude Code を使えば、自分だけのWebサイト（ホームページ）を数分で作れます。
HTML・CSS・JavaScript といったプログラミングの知識は不要。作りたいイメージを日本語で伝えるだけで、プロ品質のサイトが完成します。

---

## 必要な準備

- **Claude Code** がインストール済みであること
- **ブラウザ**（Safari, Chrome など。作ったサイトの確認用）
- （任意）**GitHub アカウント** — 作ったサイトをインターネットに公開したい場合

---

## 手順

### Step 1: 作業フォルダを作る

ターミナル（黒い画面）を開いて、以下をコピペしてください。
サイトを作るための専用フォルダを用意します。

```bash
mkdir ~/Desktop/my-site && cd ~/Desktop/my-site
```

> デスクトップに `my-site` というフォルダが作られます。

### Step 2: Claude Code を起動する

同じターミナルで、以下を入力します。

```bash
claude
```

> Claude Code が起動して、入力待ちの状態になります。

### Step 3: CLAUDE.md を作る（デザインの指示書）

Claude Code に「こんなデザインにしてね」という指示書を渡すと、仕上がりが格段に良くなります。
以下のプロンプトをコピペして入力してください。

```
以下の内容で CLAUDE.md ファイルを作成してください。

# デザインガイドライン
- HTMLファイル1つ（index.html）で完結させる
- ダークテーマ（背景は濃い色）
- メインカラー: #e8643a（暖かみのあるオレンジ）
- アクセントカラー: #3abba8（ティール）
- Google Fonts を使って、おしゃれなフォントにする
- 雰囲気: 落ち着きがありつつも、親しみやすい和モダン
- 背景にグレイン（ざらざら）テクスチャで奥行きを出す
```

> CLAUDE.md というファイルが作られます。これが今後のデザインの基準になります。
> 色や雰囲気は、お好みに合わせて自由に変えてください。

### Step 4: Plan mode で計画を立ててもらう

いきなり作るのではなく、まず「どう作るか」の計画を見てから実行すると安心です。
**Shift + Tab** を押して Plan mode に切り替えてから、プロンプトを入力します。

```
私のビジネス「手作りアクセサリーショップ Hana」の紹介サイトを作ってください。

内容:
- ショップ名とキャッチコピー（ヒーローセクション）
- コンセプト紹介（手作りへのこだわり）
- 作品ギャラリー（ダミー画像でOK）
- 料金・オーダーの流れ
- お問い合わせセクション（SNSリンク）
- フッター

デザイン:
- CLAUDE.md のガイドラインに従う
- スマホでも見やすいレスポンシブデザイン
- スクロールすると要素がふわっと現れるアニメーション
```

> Claude が計画を提示してくれます。内容を確認して、OKなら **Shift + Tab** で Plan mode を解除してから「この計画で進めてください」と伝えます。

### Step 5: 作ったサイトを確認する

生成が完了したら、以下をコピペして入力します。

```
open index.html
```

> ブラウザが開いて、作ったサイトが表示されます。

### Step 6: 修正を依頼する

気になる部分があれば、日本語でそのまま伝えるだけで直してくれます。

```
ヒーローセクションの文字をもっと大きくして、背景にぼんやり光るオーブのエフェクトを追加してください。
```

```
ギャラリーのレイアウトを2列から3列に変えて、ホバーしたときに拡大するようにしてください。
```

```
全体のフォントを「Kaisei Opti」に変えてください。
```

> 修正は何度でも無料で頼めます。納得いくまで調整しましょう。

## AIっぽくないデザインにするには

AIが作ったサイトは、何も指示しないと「どこかで見たことある」感じになりがちです。
これは、AIが「よくあるパターン」に引っ張られるためです。

### よくあるAIっぽいデザイン（NG例）

- 紫〜青のグラデーション背景
- Inter, Roboto, Arial などの無難すぎるフォント
- 中央揃えの大きい文字 + 説明文 + ボタン、という定型パターン
- 均等に色を配分した、特徴のない配色

### 対策1: `/frontend-design` スキルを使う

Claude Code には「AIっぽくならない」ためのスキルが用意されています。
プロンプトの前に `/frontend-design` と入力するだけで、個性的なデザインを提案してくれます。

```
/frontend-design

手作りアクセサリーショップの紹介サイトを作ってください。
ナチュラルで温かみのある雰囲気にしたいです。
```

> スキルのインストール方法（まだ入っていない場合）:
> ```
> npx skills add https://github.com/anthropics/skills --skill frontend-design
> ```

### 対策2: CLAUDE.md にNGリストを書く

```markdown
# デザインで避けること
- Inter, Roboto, Arial は使わない
- 紫〜青のグラデーション背景は禁止
- 中央揃えのヒーロー + ボタンの定型レイアウトは避ける
- 均等配色にしない（メインカラーを大胆に使う）
```

### 対策3: デザインの方向性を具体的に伝える

NG: 「モダンでクリーンなデザインにして」（曖昧すぎてAIっぽくなる）

OK:
```
和モダンな雰囲気で、筆文字風のフォント「Kaisei Opti」をタイトルに使い、
背景は和紙のようなテクスチャ。色は墨色（#2d2d2d）ベースに朱色（#c0392b）のアクセント。
レイアウトは左右非対称で、余白を大胆にとる。
```

> ポイント: 「和モダン」「北欧ナチュラル」「レトロフューチャー」「ブルータリスト」など、具体的な形容詞を使うとAIっぽさが消えます。

## 応用例

- 結婚式・パーティーの招待ページ
- 個人ポートフォリオ（作品集）サイト
- お店の紹介サイト（カフェ、サロン、教室など）
- サークルや趣味の活動報告ページ
- イベントのランディングページ

---

## Tips & コツ

- **`open index.html`** で何度でもブラウザで確認できます。修正→確認を繰り返しましょう
- **画像はあとから差し替え可能**。最初はダミー画像で配置だけ決めてもらうのがおすすめ
- **Tailwind CSS** を使うように指示すると、よりクオリティの高いデザインになりやすいです
  ```
  Tailwind CSS の CDN を使ってスタイリングしてください。
  ```
- **Git で保存** しておくと、いつでも前の状態に戻せるので安心です
  ```
  git init && git add -A && git commit -m "初版"
  ```
- **GitHub Pages** を使えば、作ったサイトを無料でインターネットに公開できます。Claude Code に「GitHub Pages にデプロイする手順を教えて」と聞いてみてください
