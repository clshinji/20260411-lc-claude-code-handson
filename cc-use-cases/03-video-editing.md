# 動画編集

## 概要

Claude Code を使えば、動画のカット、結合、字幕追加、フォーマット変換などを日本語で指示するだけで実行できます。
裏側では **FFmpeg** という動画編集ツールが動いていますが、難しいコマンドを覚える必要はありません。
「この動画の30秒から1分30秒だけ切り出して」と伝えるだけでOKです。

---

## 必要な準備

- **Claude Code** がインストール済みであること
- **FFmpeg**（動画を処理するためのツール）
- **編集したい動画ファイル**

### FFmpeg のインストール方法

ターミナル（黒い画面）を開いて、以下をコピペしてください。

```bash
brew install ffmpeg
```

> Homebrew（Mac用のソフト管理ツール）が入っていない場合は、Claude Code を起動して「FFmpegをインストールしたい」と伝えてください。インストール方法を案内してくれます。

インストールできたか確認するには:

```bash
ffmpeg -version
```

> バージョン番号が表示されれば成功です。

---

## 手順

### Step 1: 動画ファイルを作業フォルダに置く

```bash
mkdir ~/Desktop/my-video && cd ~/Desktop/my-video
```

編集したい動画ファイルを、デスクトップの `my-video` フォルダにコピーしてください。
Finder（ファイル管理アプリ）でドラッグ&ドロップすればOKです。

> ファイル名に日本語やスペースが含まれていると問題が起きることがあります。`input.mp4` のようにシンプルな英数字の名前に変えておくと安心です。

### Step 2: Claude Code を起動する

```bash
claude
```

### Step 3: まず動画の情報を確認する

何ができるか知るために、まず動画の中身を確認してもらいましょう。

```
input.mp4 の動画情報（解像度、長さ、ファイルサイズなど）を教えてください。
```

> 動画の長さ、画質、ファイルサイズなどが表示されます。この情報をもとに、次の編集を進めます。

### Step 4: やりたい編集を日本語で伝える

以下のプロンプト例をコピペして、必要に応じて内容を変えてください。

**動画のトリミング（一部を切り出す）**

```
input.mp4 の 0:30 から 1:30 までを切り出して、output.mp4 として保存してください。
```

**字幕を追加する**

```
input.mp4 に以下の字幕を追加して、subtitled.mp4 として保存してください。

0:00-0:05 「こんにちは！今日は手作りアクセサリーの作り方を紹介します」
0:05-0:10 「用意するものはこちら」
0:10-0:20 「まずビーズを糸に通していきます」
```

**動画をGIFに変換する**

```
input.mp4 の 0:05 から 0:10 の部分をGIFアニメーションに変換してください。
横幅480pxで、ファイルサイズが小さくなるように最適化してください。
```

**サムネイル画像を作る**

```
input.mp4 の 0:15 のフレームをサムネイル画像（thumbnail.jpg）として保存してください。
```

**複数の動画を結合する**

```
video1.mp4 と video2.mp4 と video3.mp4 をこの順番で結合して、combined.mp4 として保存してください。
```

**動画を圧縮する（ファイルサイズを小さく）**

```
input.mp4 を画質をなるべく維持したまま、ファイルサイズが半分程度になるように圧縮して、
compressed.mp4 として保存してください。
```

**動画のフォーマットを変換する**

```
input.mov を MP4 形式に変換してください。
```

### Step 5: 結果を確認する

```
open output.mp4
```

> デフォルトの動画プレイヤーで再生されます。

### Step 6: さらに編集を重ねる

満足できるまで、追加の編集を何度でも依頼できます。

```
output.mp4 の再生速度を1.5倍にしてください。
```

```
output.mp4 にBGMとして music.mp3 を追加してください。
BGMの音量は動画の音声の30%くらいにしてください。
```

## 応用例

- **SNS用の短尺動画**: 長い動画から見どころだけを切り出してリール/ショート動画に
- **動画からGIF作成**: ブログやSNSに貼れるGIFアニメーションを作成
- **BGM追加**: 無音の動画にバックグラウンドミュージックを追加
- **動画の圧縮・リサイズ**: メール添付やSNSアップロード用にファイルサイズを縮小
- **バッチ処理**: フォルダ内の複数動画を一括で変換・リサイズ

## 発展: Remotion で動画をゼロから作る

さらに進んだ使い方として、**Remotion** というツールを使えば、コードで動画をゼロから生成できます。

Remotion は React（Webアプリ開発のフレームワーク）を使って動画を作るツールです。
テキストアニメーション、画像のスライドショー、データの可視化など、プログラムで制御した動画が作れます。

```
Remotion を使って、自己紹介動画を作ってください。

内容:
- 名前: 山田花子
- 職業: ヨガインストラクター
- 趣味: 読書、旅行
- 背景: 柔らかいグラデーション
- テキストがフェードインするアニメーション
- 長さ: 15秒
```

> Remotion のスキルをインストールすると、より高品質な動画が作れます:
> ```
> npx skills add https://github.com/remotion-dev/skills --skill remotion-best-practices
> ```

> 注意: Remotion は Node.js の知識が多少必要です。まずは FFmpeg での編集から始めるのがおすすめです。

## Tips & コツ

- **FFmpegのコマンドを覚える必要はありません**。やりたいことを日本語で伝えれば、Claude Code が適切なコマンドを生成して実行してくれます
- **元の動画ファイルは上書きされません**。出力ファイル名を別にするので安心です。念のため元ファイルのバックアップを取っておくとより安心
- **まず短い動画でテスト**しましょう。大きなファイル（数GB）は処理に時間がかかります
- **「この動画の情報を教えて」** と聞くと、解像度・コーデック・長さなどを教えてくれます。トラブル時の原因特定にも便利です
- **複数の操作をまとめて依頼**できます:
  ```
  input.mp4 を以下の手順で編集してください:
  1. 0:10 から 2:00 を切り出す
  2. 字幕を追加（「チャンネル登録お願いします」を最後の5秒間表示）
  3. MP4で出力
  ```
- **動画のファイル名**は英数字のシンプルなものにしておくと、トラブルが少ないです
