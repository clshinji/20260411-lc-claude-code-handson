# チラシ作り

## 概要

Claude Code を使えば、イベント告知やお店のメニューなどのチラシをプロ品質で作れます。
HTML/CSS でデザインし、ブラウザの印刷機能で PDF に変換するシンプルな流れです。
Canva などのデザインツールを使わなくても、日本語で指示するだけで完成します。

---

## 必要な準備

- **Claude Code** がインストール済みであること
- **ブラウザ**（Safari, Chrome など。デザイン確認 & PDF変換用）

---

## 手順

### Step 1: 作業フォルダを作る

ターミナル（黒い画面）を開いて、以下をコピペしてください。

```bash
mkdir ~/Desktop/my-flyer && cd ~/Desktop/my-flyer
```

> デスクトップに `my-flyer` というフォルダが作られます。

### Step 2: Claude Code を起動する

```bash
claude
```

### Step 3: CLAUDE.md を作る（チラシの仕様書）

チラシのサイズや印刷向けの設定をまとめた指示書を作ります。

```
以下の内容で CLAUDE.md ファイルを作成してください。

# チラシ作成ルール
- HTMLファイル1つ（index.html）で完結させる
- サイズ: A4縦（210mm x 297mm）
- @media print で印刷用のページサイズを指定する
- 余白: 上下左右15mm以上（印刷時の裁ち落とし考慮）
- Google Fonts の日本語フォントを使用する
- 画像が必要な箇所は placeholder.co のダミー画像を使用する
- 背景色は印刷に適した明るいベースにする
```

> A4縦 がもっとも一般的なチラシサイズです。横向きにしたい場合は「A4横（297mm x 210mm）」に変えてください。

### Step 4: Plan mode でチラシの計画を確認する

**Shift + Tab** で Plan mode に切り替えてから、以下を入力します。

```
ヨガ教室の体験レッスン告知チラシを作ってください。

掲載情報:
- 教室名: Yoga Studio Calm
- キャッチコピー: 「心と体を整える、はじめての一歩」
- 体験レッスン日時: 毎週土曜 10:00〜11:30
- 場所: ○○市文化センター 2F 多目的ホール
- 料金: 初回無料
- 持ち物: 動きやすい服装、タオル、飲み物
- 講師紹介: 山田花子（ヨガ歴15年、全米ヨガアライアンス認定）
- 予約方法: LINE公式アカウント @yoga-calm
- QRコード用のスペース（あとで画像を差し替える）

デザイン:
- ナチュラルで落ち着いた雰囲気
- メインカラー: カーキグリーン系
- フォント: Noto Sans JP
- ヨガのポーズ写真が入るスペースを確保（ダミー画像でOK）
```

> 計画を確認して、良ければ **Shift + Tab** で Plan mode を解除し「この計画で進めてください」と伝えます。

### Step 5: デザインを確認する

```
open index.html
```

> ブラウザでチラシのデザインが表示されます。

### Step 6: 修正を依頼する

気になる部分は日本語で伝えるだけで修正できます。

```
タイトルをもっと目立つように大きくして、背景にうっすら葉っぱのイラスト（SVG）を追加してください。
```

```
料金のところを枠で囲んで、もっと目立つようにしてください。
```

### Step 7: PDF に変換する

デザインが完成したら、PDFファイルにして印刷できるようにします。

**方法1: ブラウザの印刷機能（最も簡単）**

1. ブラウザでチラシを開いた状態で **Cmd + P**（印刷）
2. 送信先を **「PDFとして保存」** に変更
3. 用紙サイズが **A4** になっていることを確認
4. **「余白」を「なし」** に設定
5. 「保存」をクリック

> これだけで印刷可能なPDFファイルが完成します。

**方法2: Claude Code にPDF生成を頼む**

```
このチラシを puppeteer を使ってPDFファイルとして出力するスクリプトを作って実行してください。
A4サイズ、余白なしで出力してください。
```

> Node.js がインストールされている必要があります。方法1で十分な場合はこちらは不要です。

## AIっぽくないデザインにするには

チラシでも、AIに任せきりだと「テンプレ感」が出がちです。以下の工夫で差をつけましょう。

### よくあるAIっぽいチラシ（NG例）

- ありきたりなレイアウト（上にタイトル、中央に写真、下に情報）
- 紫〜青のグラデーション
- 均等に並んだ要素で余白が中途半端
- 特徴のない色使い

### 対策1: `/frontend-design` スキルを使う

```
/frontend-design

ヨガ教室の体験レッスンチラシを作ってください。
ナチュラルで温かみのある雰囲気、手作り感のあるデザインにしたいです。
```

> スキルのインストール方法（まだ入っていない場合）:
> ```
> npx skills add https://github.com/anthropics/skills --skill frontend-design
> ```

### 対策2: 具体的なデザイン方向性を伝える

NG: 「おしゃれなチラシにして」

OK:
```
ナチュラル系のデザインで、リネン素材のようなテクスチャを背景に使い、
手書き風のアクセントを入れてください。色はカーキとベージュをベースに、
テラコッタカラーをアクセントに。フォントは「M PLUS Rounded 1c」で柔らかい印象に。
レイアウトは上下2分割ではなく、斜めの区切り線で動きを出してください。
```

### 対策3: CLAUDE.md にNGリストを書く

```markdown
# デザインで避けること
- テンプレートのような定型レイアウトは避ける
- Inter, Roboto, Arial は使わない
- 紫〜青のグラデーションは禁止
- 要素を均等に並べず、強弱をつける
```

- お店のメニュー表（カフェ、レストラン）
- イベント・セミナーの告知ポスター

## Tips & コツ

- **ブラウザの印刷（Cmd + P → PDF保存）が最も簡単なPDF変換方法**です。まずはこれで試しましょう
- **A4サイズの指定**は `width: 210mm; height: 297mm;` で行います。CLAUDE.md に書いておけばClaude が自動的に適用してくれます
- **日本語フォント** のおすすめ:
  - `Noto Sans JP` — 読みやすいゴシック体
  - `Noto Serif JP` — 上品な明朝体
  - `M PLUS Rounded 1c` — 柔らかい丸ゴシック
  - `Kaisei Opti` — 和風でモダンな印象
  - `Zen Maru Gothic` — 親しみやすい丸ゴシック
- **画像はあとから差し替え可能**。まずダミー画像で配置を決めてから、実際の写真に入れ替えましょう
- **QRコードの画像**は、QRコード生成サイトで作成して差し替えるのがおすすめです
- **`pdf` スキル** をインストールすると、PDF操作がさらに便利になります:
  ```
  npx skills add https://github.com/anthropics/skills --skill pdf
  ```
