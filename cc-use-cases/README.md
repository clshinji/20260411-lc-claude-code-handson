## 事例の読み方

各事例は以下の構成になっています:

1. **概要** — 何ができるのか
2. **必要な準備** — 事前に用意するもの
3. **手順** — Step by Step で進める具体的な作業
4. **応用例** — 「こんなことにも使える」アイデア
5. **Tips & コツ** — 知っておくと便利なポイント

プロンプト例（Claude Code に入力する文章）はすべてコピペできる形で掲載しています。

## 事例を追加するには

1. `NN-slug.md`（例: `04-sns-automation.md`）の形式でファイルを作成
2. 上の「事例一覧」テーブルに1行追加
