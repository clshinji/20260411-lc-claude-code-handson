## Claude で提供されるアプリの違い

なんとなく、凄さを伝えたところで、 **Claude** で使えるサービスを紹介（おさらい）しましょう。

- Claude では次は3つのツールが使えます
  - 1. Claude (Claude AI)
    - ChatGPTやGeminiみたいに、チャットができます
    - Claudeは、インタラクティブなグラフ（＝動くグラフ）をチャットの中に埋め込んでくれる機能もあります
  - 2. Claude Code
    - スーパーエンジニアのように、コーディングができます
    - AIエージェントが自律的にコードをかいて、パソコンの操作をすることもできます
    - 調べ物をしたり、ファイルの操作を自動化したり、Webページやアプリを作ったり、本格的なシステムを作ることもできます
  - Claude Cowork
    - まだ **research preview(＝研究)** 段階の機能
    - Claude Code と同じように、全自動でタスクを実行してくれる
    - タスクを指示したら、成果物を作るところまで実行して、結果を報告してくれます。つまり、 **※途中の承認は無しでやり切ってくれる！**

## Claude Code が使える場所

とても便利そうな **Claude Code** ですが、使える場所がいくつかあります。

1. **ターミナル** :通称黒い画面
2. **VSCode** :エンジニア御用達の高性能なメモ帳
3. **アプリ** :Claude公式のアプリ

この3つの違いを比較すると、こんな感じです。

| 使用場所 | 良いポイント | デメリット |
| --- | --- | --- |
| **ターミナル** | 最新機能がいち早く登場する！ | ただしコマンド操作(CLI)に慣れていないと、とっつきにくい。 |
| **VSCode** | Claude Code が変更したファイルを確認しやすい | ターミナル版よりは機能が限定的なことも… |
| **アプリ** | チャットで慣れ親しんだデザイン | ターミナル版よりは機能が限定的なことも… |

### はじめにターミナルの操作を練習します

ターミナルを操作するのが初めてな方向けに、ターミナルの操作を少し練習してみましょう。（ちょっとエンジニアっぽい雰囲気を体験）

- ターミナルの基本的な使い方 (Claude Code もだいたい同じ)
 - マウス操作は ほぼ 使えません...
 - 1つ前のコマンドをもう一度使いたいときは **↑ キー** を押します
 - **(重要)** 処理を中断するときは **control + c**  ※ `Command` ではなく `Control` 
 - 終了するときは **赤い x ボタンでウィンドウを閉じる** でもOK
- ターミナルを使うときの注意点
 - **Enter** キーを押すときは慎重に。ちょっと怖いコマンドも実行できてしまうかも？
 - **半角/全角 どっち？** かを意識しましょう。
   - 通常は **半角モード** にします。日本語を入力するときだけ 全角モード にします。
- ターミナルの便利な使い方
 - (Macの場合)フォルダを **Dockのターミナルのアイコンにドラッグ＆ドロップ** すると、そのフォルダでターミナルを開けます。
 - 作業したいフォルダに移動する方法
   - `cd ` (シー ディー スペース) と入力した後に、 **ターミナルの画面にフォルダをドラッグ＆ドロップ** して **Enter** で決定 -> フォルダを移動できます

### Claude Code を使ってみよう

Claude Code に作業してもらうフォルダを、ターミナルで開きます。

> (おさらい) フォルダをターミナルで開く方法

### 基本的な使い方

`/` から始まるコマンドで、Claude Codeに特別な指示が出せます。

試しに `/` を押すと、チャット欄の近くに候補が表示されます。↑↓キーで選択できるようになっています。

(下の方に英語で操作方法の説明が出ています)

```
(例)
/clear : 会話をリセットする
/exit  : 終了してターミナルに戻る
 ↑ /quitでもOK

/model : Claude のAIモデルを切替えられる
/config: 設定を確認できる
/usage : 使用量を確認できる
```

> もしものときのコマンド
> esc キー →前の画面に戻る
> ctrl+c  →操作を中断する
> /resume →前のチャットの履歴から再開する
> /login  →ログイン
> /logout →ログアウト

### Claude Code のセキュリティ対策

Claude Code を起動した状態で、次のコマンドを実行しましょう。

```
/sandbox
```

安全に利用できるように、モードを選択します。

```
※ どちらか好みの方を選択してください
  1. auto-allow モード: 安全なコマンドは実行してOK
  2. regular モード: 安全なコマンドでも確認してから実行
```
